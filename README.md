## ✨ Vision
This portfolio is more than just a resume; it's a sensory experience. Built with a focus on **Cinematic Motion Design**, **Premium Glassmorphism**, and **Deeply Integrated AI**, it reflects the technical rigor and design-first mindset of its creator.

## 🧠 Integrated AI Assistant: "Shashwat AI"
The site features a unified, premium AI interface (merged text + voice) that embodies my professional persona—witty, sharp, and technically grounded.
- **Natural Language**: Detects and responds in English, Hindi, or Gujarati (Hinglish/Gujlish).
- **Voice Synthesis**: Real-time audio visualization and human-like speech responses.
- **Direct OpenAI Integration**: Powered by GPT-4o-mini via Supabase Edge Functions.

## 🛠 Technical Stack

### **Frontend & Motion**
- **Core**: React 18 + TypeScript + Vite
- **Styling**: Tailwind CSS (Glassmorphism & Obsidian Dark Theme)
- **Animations**: Framer Motion (Staggered reveals, peer-blur effects) + GSAP
- **3D Elements**: Three.js / React Three Fiber
- **Icons**: Lucide React

### **Backend & Intelligence**
- **Infrastructure**: Supabase (PostgreSQL + Edge Functions)
- **AI Engine**: OpenAI API (Direct Integration)
- **Voice**: Web Speech API / SpeechSynthesis
- **Security**: Hardened CORS, X-Frame-Options, and sanitized Prompt Engineering.

## 🚀 Quick Start (Local Setup)

1. **Clone & Install**
   ```bash
   git clone https://github.com/Shashwatology/Shashwat-Upadhyay-.git
   cd Shashwat-Upadhyay-
   npm install
   ```

2. **Environment Setup**
   Create a `.env` file in the root:
   ```env
   VITE_SUPABASE_URL=your_supabase_url
   VITE_SUPABASE_ANON_KEY=your_supabase_anon_key
   ```

3. **Deploy Edge Functions** (Optional)
   ```bash
   npx supabase functions deploy chat
   ```

4. **Launch**
   ```bash
   npm run dev
   ```
